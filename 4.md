**博弈差核心解读**

- 独赢盘去水后土耳其**47.2%**，而Opta算力给出**49.4%**，差值+2.2pp，是本场博彩公司与模型落差最大的市场——博彩公司轻微低估土耳其。
- 巴拉圭最近10场世界杯赛事中，9场进球不足2.5球，但博彩公司大球隐含概率高达53.6%（去水后），明显偏高——这是**博彩公司在大球方向做热的信号**。
- 全场让球盘中博彩公司去水后巴拉圭+0.5胜出概率**52.2%**，代表他们认为土耳其净胜1球以上仅47.8%，比Opta胜率低约1.6pp，是轻度压制土耳其方向。

**盘口异动判断**：博彩公司真实防范重心在**小球**（调低小球赔率至2.08引流大球），意图将公众资金引向大球。

### ⏳ 三、 Game State 推演

- **僵局期（0-20'）**：土耳其必然主动压制，高位反抢争夺中场主导权。巴拉圭面临淘汰压力后防线更加谨慎，倾向整体退守半场等待定点球机会。值得注意的是土耳其在大赛上已连败9场中的13场，心理压力显著，开局容易焦虑。

- **破局点**：最大可能的进球方式为**土耳其定位球或巴拉圭防线错位**。居勒或恰尔汗奥卢的任意球威胁是核心。居勒一旦首发出场，土耳其进攻输出将成倍提升。

- **领先后（若土耳其进球）**：土耳其大概率继续控球消耗，压缩巴拉圭反击空间。巴拉圭前场只有萨纳比亚和阿尔米龙具备威胁，体系不足以维持持续攻势。

- **衰竭期（75'-90'）**：土耳其已连续两场在上半场丢球，下半场体能反升。巴拉圭方面若落后，75分钟后防线全面拉开，尾声阶段存在补时进球风险，但绝杀性反击概率不超过15%。

**📝 依据与计算推导**

- **全场独赢/全场让球（一级Opta数据）**
  Opta超算：土耳其胜49.4% × 欧赔2.03 = **EV 1.003**
  博彩公司独赢去水后47.2%，Opta给49.4%，差值+2.2pp，薄利正期望但不足以构成强势✅。评级：⚠️

- **全场大小球（三级泊松推导⚠️ —— 世界杯仅1场数据，不足5场样本，强制封顶⚠️）**

  ```
  λ_土耳其 = (土耳其场均xG产出1.40 + 巴拉圭场均xG失球1.35) ÷ 2 = 1.375
  λ_巴拉圭 = (巴拉圭场均xG产出0.47 + 土耳其场均xG失球1.00) ÷ 2 = 0.735
  总λ = 2.11

  P(总进球=0) = e^(-2.11) = 12.1%
  P(总进球=1) = 12.1% × 2.11 = 25.6%
  P(总进球=2) = 12.1% × 2.11² / 2 = 27.0%
  P(总进球≥3) = 35.3%

  亚洲盘2/2.5结算规则：
  押小：≤1全赢(37.7%)，=2半赢(27.0%)，≥3全输(35.3%)
  EV(押小) = 37.7%×1.08 + 27.0%×0.54 + 35.3%×(-1) = 0.406 + 0.146 - 0.353 = +0.199 → EV 1.198
  EV(押大) = 35.3%×0.80 + 27.0%×0.40 + 37.7%×(-1) = 0.283 + 0.108 - 0.377 = +0.014 → EV 1.014
  ```

  > 定性支撑：巴拉圭最近10场世界杯9场不足2.5球历史纪录；土耳其首战0.05 xG/射的极低转化率；盘口博彩公司大球方向做热（大球隐含概率53.6%，远超模型推算35.3%）。

- **半场让球（0/0.5）（三级推导，样本不足，⚠️仅供参考）**
  上半场λ_土 = 1.375 × 0.45 = 0.619；λ_巴 = 0.735 × 0.45 = 0.331
  土耳其上半场净胜概率估算约32%，平局约43%，巴拉圭领先约25%
  押巴拉圭（不输）方：EV = 68% × 1.77（部分还款结构）= 约1.12⚠️
  土耳其上半场历史连续两场丢球，开局主动进攻但转化极差，上半场净胜方向EV偏弱。
