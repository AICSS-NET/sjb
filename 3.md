### 📊 二、 盘口测谎与概率博弈
1. **基线**：独赢盘隐含概率 `(1/1.10)=90.9%`，博彩公司抽水率 = `(90.9% + 9.09% + 4.35%) - 1` = 4.34%。去除抽水后，巴西真实胜率估值为 87.1%，该数据与 Opta 模型超算预测（87%赢球概率）严丝合缝。
2. **博弈差**：胜负盘基线虽吻合，但**让球盘与大小球盘**存在严重偏离。全场让球线从初始的巴西让3球，断崖退至即时盘的2.5球；总进球数盘口从4球急退至3.5球。此为盘口博弈中差值最大的选项，机构在利用退盘掩饰巴西“赢球输盘”的极高概率。
3. **盘口异动**：强队面对弱队遭遇连续降线退盘，戳穿了博彩公司的真实防范重心：机构深刻忌惮巴西首场“得势不得分”的疲软态势（514次传球仅换来1球），且顺应了海地首战展现出的禁区压迫力（22次对方禁区触球，与巴西首轮持平）。让步缩水实质为庄家在基本面压制下的主动避险。

### ⏳ 三、 Game State 推演
- **僵局期**：前30分钟海地体能处于峰值，由德尔克鲁瓦领衔的5后卫体系将极大压缩维尼修斯和拉菲尼亚的冲刺空间。巴西缺乏内马尔前场接应串联，极易陷入外围横传无效倒脚的陷阱（高控球率、低期望进球）。
- **破局点**：最可能的进球方仍为巴西，破局方式大概率为定位球、远射折射或边路强行起球制造混战。海地首发门将普拉西德护框及密集防守体系将剥夺巴西中路直接渗透打穿的概率。
- **领先后**：安切洛蒂的务实战术体系决定了比赛的收官走势。一旦巴西取得1至2球领先，球队更倾向于通过吉马良斯和帕奎塔在后场进行倒脚控场消耗，绝不强行提速刷净胜球以备战末轮对苏格兰的硬仗。
- **衰竭期**：70分钟后海地因长期被动防守导致体能透支，防线横向移动变慢。巴西替补群（库尼亚/罗德里戈等）有概率利用转换空间偷袭得手，但在剩余比赛时间维度下，打穿净胜3球让步的容错空间已被彻底封死。
