### 🔍 五、+EV策略

> **数据层级声明**：独赢采用Opta一级数据；让球/大小球缺乏Opta覆盖，使用三级泊松推导（样本仅世界杯1场+热身赛，所有推导结果强制封顶⚠️）。

**λ推导基础**：
```
λ_巴西（保守）= 3.0　｜　λ_巴西（激进）= 3.5
λ_海地 = 0.7
λ_total（保守）= 3.7　｜　λ_total（激进）= 4.2
```
激进取值依据：三场H2H平均5.67球，且全部超过3.5球；巴西需净胜球表现；激进λ更符合历史规律。

**📝 依据与计算推导**

- **半场让球（巴西-1/1.5）**：
```
λ_巴西_HT = 3.0×45% = 1.35
λ_海地_HT = 0.7×45% = 0.315

P(净胜≥2) ≈ 32%（全胜）
P(净胜=1) ≈ 31.8%（半胜）

加权胜率 = 32% + 31.8%×0.5 = 47.9%
EV = 47.9% × 2.07 = 0.992 → ⚠️（三级封顶）
```

- **全场让球（巴西-2.5）**：
```
P(净胜≥3)保守 = P(B=3+,H=0)+P(B=4+,H=1)+…≈43%
P(净胜≥3)激进（λ=3.5）≈ 55%

EV保守 = 43%×1.84 = 0.791 → ❌
EV激进 = 55%×1.84 = 1.012 → ⚠️（三级封顶）
```
博彩公司将覆盖率定在52.4%，高于保守模型10点。若巴西真实动机（净胜球需求）驱动攻势比模型更强，激进端估算更可信。

- **全场独赢（巴西）**：`EV = 87.3%（Opta一级）× 1.10 = 0.960` → 赔率已被压至极限，独赢无价值。

- **全场大小（3.5大）**：
```
Poisson(λ=3.7): P(≥4) = 50.6%，EV = 0.926 → ❌
Poisson(λ=4.2): P(≥4) = 60.5%，EV = 1.108 → ⚠️（三级封顶）
```
H2H历史三场均超3.5球支持激进端；然而博彩公司已将大球定价在去水52.6%，溢价空间压缩。
