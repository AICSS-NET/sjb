### ⏳ 三、Game State 推演

**僵局期（0–20分钟）：** 美国势必高位压迫，上场迫使对手产生16次高位断球，全届赛事仅西班牙超越。澳大利亚将固守阵型，以门将帕特里克·比奇为核心抵抗首波冲击。

**破局点：** 巴洛冈为本场xG模型赋予28.4%射手概率，为全场最高，是美国最可能的破门点。但澳大利亚惯用低位深防，让美国在外围传射，自身通过反击寻求机会——这正是1-1或1-0低比分剧本的温床。

**领先后：** 若美国先入球，澳大利亚必须出来对攻，反而为美国创造更多反击空间，但也令比分线更扑朔。若澳大利亚通过定位球或反击意外领先，美国需全力压攻，比赛将进入高度混乱期。

**衰竭期（75分钟+）：** 澳大利亚对阵土耳其仅28%控球，体能储备充分，尾声抗压能力优秀。美国若未能扩大至两球以上，最终比分落在**1-0或2-1**的概率极高。

**📝 依据与计算推导**

**全场让球 澳大利亚+1（一级Opta数据）：**
美国胜率58.5%中，赢球2球以上的估算概率约32%（结合xG回归：美国本场真实xG产出上限受澳大利亚防线压制）。澳大利亚+1覆盖概率 = 平局(20.9%) + 澳洲胜(20.6%) + 美国仅胜1球(约26.5%) ≈ **68.0%**。
`EV = 68.0% × 1.85 = 1.258 ✅`
博彩公司去水隐含美国覆盖47.6%，与模型32%相差超15%，是本场最大错误定价。

**全场独赢 澳大利亚（一级Opta数据）：**
Opta真实胜率20.6%，博彩公司去水隐含17.5%，价值差3.1%叠加5.50赔率：
`EV = 20.6% × 5.50 = 1.133 ✅`

**全场大小球（三级泊松推导）：**
数据源：美国vs巴拉圭 xG产出=1.42；澳大利亚vs土耳其 xG产出=1.18，xG失球=1.36
```
λ_美国 = (1.42 + 1.36) ÷ 2 = 1.39
λ_澳大利亚 = (1.18 + 0.50*) ÷ 2 = 0.84
（*美国对阵巴拉圭xG失球估算约0.50）
```
泊松联合分布下 P(总进球≤2) = **61.4%**
`EV(小球) = 61.4% × 1.97 = 1.210` → 三级推导强制封顶 ⚠️

**半场让球 美国 -0/0.5（二级推算）：**
依据：美国近12场对阵世界杯球队中9场上半场有进球，半场领先概率约48%，半场平概率约37%，澳大利亚半场领先约15%。四分球EV计算：
`EV = 0.48×1.75 + 0.37×0.5 = 0.840 + 0.185 = 1.025` ⚠️
